# ✂️ Barbería García - Grooming & Style

Proyecto web profesional desarrollado para Barbería García, enfocado en el arte de la barbería clásica y tendencias modernas de cuidado masculino.

🚀 Características

* **Diseño Premium:** Interfaz elegante con estética Dark Mode y acentos dorados.
* **Navegación Intuitiva:** 5 secciones optimizadas (Inicio, Servicios, Expertos, Lookbook y Tienda).
* **Conversión Directa:** Botón flotante de WhatsApp y botones de compra integrados para agendamiento rápido.
* **Responsive Design:** Experiencia fluida adaptada a smartphones, tablets y laptops.
* **Arquitectura Limpia:** Uso de HTML5 semántico y variables CSS para fácil mantenimiento.

🛠️ Tecnologías utilizadas

* HTML5 (Estructura semántica avanzada)
* CSS3 (Custom Properties, Flexbox y Grid)
* FontAwesome (Iconografía profesional)
* Google Fonts (Poppins)
